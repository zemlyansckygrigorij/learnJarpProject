Данный проект предназначен для отработки навыков в следующих технологиях
PostgreSQL, Spring Boot, Spring JPA,WebSphere MQ, Apache ActiveMQ,Junit, TestNG,Kafka, Redis

Описание:
Данное приложение
1) Извлекает поступившие данные из документов в формате XML, XSD, XSL
2) Сохраняет полученные данные в СУБД PostgreSQL
3) Отправляет полученные данные на другие микросервисы  используя  технологии WebSphere MQ, Apache ActiveMQ

This project is designed to practice skills in the following technologies
PostgreSQL, Spring Boot, Spring JPA, WebSphere MQ, Apache ActiveMQ, Junit, TestNG, Kafka, Redis

Description:
This application
1) Extracts the received data from documents in XML, XSD, XSL format
2) Saves the received data in the PostgreSQL DBMS
3) Sends the received data to other microservices using WebSphere MQ, Apache ActiveMQ technologies
